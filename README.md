# BytePizza Ordering System

BytePizza is a software engineering team project focused on designing and developing a digital pizza ordering application. The goal of the system was to simulate a real world ordering experience where users could browse menu items, customize selections, and place orders through an intuitive interface.

This project was completed as part of a collaborative academic software development team.

---

## Project Overview

The application was designed to demonstrate core software engineering principles including system design, object oriented programming, user interface planning, and team based development workflows.

Key features included:

- Menu browsing and item selection
- Order customization
- Pricing calculations
- User interaction flow design
- Structured system architecture planning
